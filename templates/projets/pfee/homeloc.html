Présentation du projet

Homeloc, première solution de multidiffusion d'annonces de location de vacances, souhaite baser son développement sur la création d'une BDD touristique géolocalisée, à partir de données publiques et privées, continuellement enrichies et normalisées. Le projet s’inscrit des étudiants dans cette optique.

Afin d’éviter aux utilisateurs de devoir renseigner de trop nombreuses informations, l’idée est de récupérer le plus de données pertinentes grâce à l’Open Data. Le développement a consisté en une interface web afin de permettre aisément à Homeloc de contrôler les différents jeux de données recueillis. La base de données ainsi créée est accessible par Homeloc et leurs partenaires grâce à un webservice.
