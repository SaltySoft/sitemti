Run or Die

Run Or Die est un jeu 3D de type Temple Run réalisé réalisé à l’aide d’Unity 3D, jouable sous Windows ou Mac. Le but est de survivre le plus longtemps possible en évitant des zombies dans une atmosphère terrifiante.

Gravity Game

Gravity Game est un jeu 2D entièrement réalisé en HTML5. Le joueur évolue dans un système solaire L'objectif est de créer la vie sur des planètes en y déposant les ingrégients nécessaires, et de leur apporter de l'énergie en ravivant l'étoile.
