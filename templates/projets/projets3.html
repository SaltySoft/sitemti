MTIDisquaire

Cette application permet la gestion de bibliothèque CD qui permet de :

  • Ajouter, modifier, supprimer des CD;
  • Gérer les artistes des CD;
  • Gérer les albums par artiste;
  • Gérer les albums par genre;
  • Modifier les stocks;
  • Rechercher les CD.
