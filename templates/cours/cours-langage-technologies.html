• Environnement .NET et C#

    Cette matière enseigne le maniement des outils de développement du framework .NET de Microsoft en C#, en allant de la création d'un serveur web à l'édition de tests unitaires.

  • Java EE

    Cette matière enseigner le développement JAVA grâce aux outils JEE et aux différents frameworks disponibles: utilisation de Spring, Hibernate, JPA

  • Initiation au langage et Frameworks PHP

    Ce cours enseigne les bases du développement PHP en prenant en compte la sécurisation d'une application web-serveur, et en présentant de nombreux outils disponbiles.

  • Javascript

    Ce cours enseigne aux étudiants de MTI les bases du langage Javascript, et leur permet de développer un projet libre dans ce langage.

  • Développement d'un Jeu Vidéo

    Cette matière enseigne aux étudiants de MTI les bases du développement de jeu vidéo, avec la présentation de différentes techniques. Le développement d'un jeu vidéo du début à la fin permet aux étudiants d'appréhender une grande partie de ces concepts.

  • Intégration HTML/CSS/JavaSript orienté design

    Cette matière enseigne aux étudiants l'importance du design dans le développement d'un projet web, et leur apprend à manier des outils comme Photoshop

  • Développement mobile iOS

    Cette matière permet aux étudiants de MTI de découvrir le développement mobile iOS. Un projet libre est également à développer pour l'évaluation.
